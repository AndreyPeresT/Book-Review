-- HOW TO START USING VS.CODE --

1. Open the BOOK REVIEW folder with the vs-code and open the bash terminal and use "npm install" to install all npm components used in this project.

2. Inside this folder I created a SQL file with the code of the table that i used to insert and query the data from the website.

3. After all past steps are completed, in the bash terminal use this line "node index.js".
